    brett_projekt v0.0.1 BrettProjekt.GameManager

    Provides state and functions for handling Game instances.

    The GameManager is implemented as an Agent holding different Game instances. These can be accessed by convenient functions.

    It also provides functions for generating unique game-identifiers.
